Dice Rolling Simulator

Type: Project • Year: 2025 • Status: In Progress

What this project does

The Dice Rolling Simulator is a web app built with EJS, Node.js, Express, and Bootstrap that lets you roll between 1 and 6 dice and instantly see each result (1–6). It includes a planned Statistical Analysis section to visualize distributions for a predefined number of rolls and dice, as well as the distribution of the sum of dice values, highlighting the tendency toward a normal-like distribution as trials increase.

  • Roll 1–6 dice and view each outcome individually
  • Clean, responsive UI with Bootstrap
  • WIP Statistical charts for outcomes and sums (distribution analysis)
  • WIP Parameters for number of trials and dice count

Tech stack

EJS CSS3 JavaScript Node.js Express Bootstrap

Role & challenges

I built the full stack: routes, EJS views, and UI. The current challenge is designing a simple, fast pipeline to generate large batches of simulated rolls and plot them efficiently on the client without blocking the UI.

  • Keeping the server’s API minimal and the client responsive
  • Producing accurate frequency distributions for outcomes and sums
  • Preparing the charting layer to support larger trials smoothly

What I learned

  • Structuring small Express apps with clean, focused routes
  • Templating with EJS for reusable UI blocks
  • Designing UIs for simulation tools with clear parameters and feedback

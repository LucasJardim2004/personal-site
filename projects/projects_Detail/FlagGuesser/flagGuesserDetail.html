Flag Guesser

Type: Game • Year: 2025 • Status: Completed

What this project does

Flag Guesser is a quiz game where a flag is displayed and the user types the country’s name. The input is validated against a PostgreSQL database — if it matches (with case and accent normalization), it counts as correct and moves to the next flag.

  • Random selection of flags from the database
  • Server-side validation with normalized matching (case & accent insensitive)
  • Immediate feedback (correct/wrong) and score tracking

Tech stack

EJS CSS3 JavaScript Node.js Express PostgreSQL

Role & challenges

I implemented the end‑to‑end flow: UI, API, and data layer. The biggest challenge was normalizing user inputs (accents, spaces, name variations) and ensuring efficient database matching (e.g., ILIKE + normalization functions) without compromising performance.

What I learned

  • Designing a simple API in Express with validation routes
  • Parameterized queries in PostgreSQL
  • String normalization (casefolding, accent removal) and use of ILIKE
  • Basic indexing and performance considerations in text matching

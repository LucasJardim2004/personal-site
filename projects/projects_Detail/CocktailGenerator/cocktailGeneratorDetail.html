Cocktail Explorer

Type: Web App • Year: 2025 • Status: Completed

What this project does

Cocktail Explorer is a web app that fetches cocktail recipes from TheCocktailDB. You can get a truly random drink or search using multiple filters (name, first letter, ingredient, alcoholic type, category, and glass). Results are combined server‑side and a matching cocktail is selected and displayed with ingredients and instructions.

  • Random cocktail endpoint using the API
  • Advanced filter form (name/letter/ingredient/alcoholic/category/glass)
  • Server-side intersection of results from multiple API endpoints
  • Detailed view with image, ingredients and step-by-step instructions
  • Graceful handling of empty states and API errors

Tech stack

EJS Node.js Express Axios Bootstrap 5 TheCocktailDB API

Role & challenges

I built the server routes, views, and client flow. The tricky part was merging different API queries (search and filter endpoints), intersecting their results by ID, and then fetching details efficiently. I also focused on robust error handling and UX for cases with no matches.

What I learned

  • Designing Express routes with async/await and error handling
  • Working with third‑party REST APIs and Axios
  • Combining and intersecting datasets on the server
  • EJS templating patterns and partials
  • Building accessible forms and clean UI states

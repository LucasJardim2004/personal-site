Simon Game

Type: Game • Year: 2024 • Status: Completed

What this project does

Simon Game is an interactive memory game. The app generates a growing sequence of tones/colors that the player must repeat in the correct order. Each round increases difficulty by adding one new step.

  • Dynamic sequence generation
  • Sound + visual feedback per button
  • Score tracking, highest streak and restart flow

Tech stack

HTML5 CSS3 JavaScript Bootstrap 5

Role & challenges

I designed and built the full game loop and UX. The main challenge was balancing timing between audio/visual cues and user input to keep it responsive and fun.

What I learned

  • Event handling and debouncing user input
  • Managing state and difficulty scaling
  • Organizing JS in small, testable functions

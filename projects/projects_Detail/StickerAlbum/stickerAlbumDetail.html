Euro 2024 Sticker Album

Type: University Web App • Year: 2024 • Status: Completed

What this project does

Euro 2024 Sticker Album is a full‑stack web app that recreates the fun of collecting football stickers. Built for the course Web Development and Mobile Applications, it includes user authentication (login & sign‑up), a homepage, a forum for discussions, a store to buy random sticker packs, a personal album that displays collected stickers, and a user profile page.

  • Secure authentication and session handling
  • Forum for users to communicate and trade ideas
  • Store with random pack opening animation and inventory updates
  • Album grid that shows owned / missing stickers
  • User profile with basic settings and collection stats

Tech stack

Django Python React HTML5 CSS3 Bootstrap 5 SQLite

Role & challenges

This was a pair project. I worked across the stack: Django views/URLs/models for auth, shop and album logic; React components for pack creation, album grid and forum threads; and Bootstrap UI polish. Key challenges included keeping React state in sync with Django sessions, validating purchases and inventory updates automatically, and designing a simple schema that keeps pack opening fair and reproducible.

What I learned

  • Django auth flows (sign‑up/login), forms and messages
  • Connecting React components to Django views/endpoints
  • Reusable UI with Bootstrap 5 and component composition
  • Using DB Browser for SQLite for quick inspection/debugging

Pong

Type: Game • Year: 2025 • Status: Completed

What this project does

Pong is a browser game built with the HTML5 Canvas. You control the left paddle with the mouse (or touch on mobile) while a simple AI controls the right paddle. The ball bounces with realistic angles based on the hit position, speeds up on rallies, and resets after a point.

  • Smooth mouse & touch controls for the player paddle
  • AI opponent that tracks the ball with clamped speed
  • Collision detection for paddles and walls with angle variation
  • Progressive ball speed increase on each successful hit
  • Lightweight game loop using requestAnimationFrame

Tech stack

HTML5 Canvas CSS3 JavaScript

Role & challenges

I implemented the rendering loop, input handling, physics, and AI. The main challenges were tuning paddle AI to feel fair, adding realistic bounce angles (based on impact position), and preventing tunneling at higher speeds.

What I learned

  • Building a clean requestAnimationFrame game loop
  • Basic 2D collision detection and response
  • Input handling for mouse and touch
  • Simple game AI and difficulty balancing
